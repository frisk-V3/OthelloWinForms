# OthelloWinForms
C++/SFML 版をベースに、C# WinForms で完全再構築した CPU 対戦オセロです。  
外部ライブラリや NuGet パッケージを一切使用せず、.NET 標準の WinForms のみで動作します。

---

## 特徴

- CPU 思考アルゴリズム（minimax + αβ枝刈り）を実装
- WinForms の Graphics API のみで描画
- 依存ライブラリなし、2ファイル構成で完結
- 先攻・後攻の選択が可能
- 2クリック方式の直感的な操作
- スコア表示、パス処理、勝敗判定を実装

---

## 実行方法

1. Visual Studio 2026 で新規 WinForms プロジェクトを作成  
2. `Program.cs` と `MainForm.cs` を上書き  
3. ビルドして実行

外部 DLL や NuGet パッケージは不要です。

---

## 操作方法

### タイトル画面
- SPACE キーで開始

### 先攻/後攻選択
- 1キー または ボタン … 先攻（黒）
- 2キー または ボタン … 後攻（白）

### ゲーム中
- 1クリック目 … 置きたいマスを選択
- 2クリック目 … 同じマスをクリックすると確定
- 無効な場所をクリックすると「できません」と表示

---

## CPU 思考アルゴリズム

CPU は以下の手法で最善手を探索します。

- minimax（深さ 2）
- αβ枝刈り
- 重み付き評価関数（角・辺を重視）

C++ 版のロジックを忠実に C# へ移植し、  
WinForms でも高速に動作するよう最適化しています。
